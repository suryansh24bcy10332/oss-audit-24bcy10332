## Chosen Software
Git – Distributed version control system.

## Scripts

### Script 1 – System Identity Report
This script displays system-level information such as kernel version, logged-in user, uptime and Linux distribution. It showcases the use of variables, command substitution, and formatted output.

### Script 2 – Package Inspector
This script checks if git is installed on the system and prints its version. It uses conditional statements and case handling.

### Script 3 – Disk Auditor
This script loops through important directories and displays their permissions and disk usage.

### Script 4 – Log Analyzer
This script scans a log file and counts the occurrences of a keyword, i.e., "error".

### Script 5 – Manifesto Generator
This script takes input from the user and generates a personalized open-source philosophy statement.
